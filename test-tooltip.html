HTTP Sources (ROCm / Intel XPU)

HTTP connections require the monitoring agent to be installed on the remote endpoint.

Quick Setup (Linux):
1. Clone this repo: git clone https://github.com/riversde/GPU-AI-Monitor.git
2. Install Flask: pip3 install flask
3. Ensure GPU CLI tools are installed:
  • AMD ROCm: amd-smi / rocm-smi
  • Intel XPU: xpu-smi / xpumanager
4. Run the agent: python3 agent/monitor_agent.py --type rocm
  (or --type xpu for Intel, or omit for auto-detect)
5. Ensure the firewall allows inbound connections on the chosen port

Standalone binary (no Python needed):
Build with: bash scripts/build_linux.sh → outputs dist/monitor_agent
Deploy and run: ./dist/monitor_agent --type rocm --port 5900

The agent exposes REST APIs at /api/rocm and /api/xpu that this dashboard polls for telemetry data.
