HTTP Sources (ROCm / Intel XPU)
HTTP connections require the monitoring agent to be installed on the remote endpoint.

Setup:
1. Install Python 3.x on the remote machine
2. Clone the agent: git clone https://github.com/riversde/rocm-monitor-agent.git
3. Install dependencies: pip install flask
4. Install amd-smi (ROCm) or xpumanager (Intel XPU)
5. Run the agent: python rocm_agent.py (default port 5900)
6. Ensure the firewall allows inbound connections on the chosen port

The agent exposes a REST API at /api/rocm that this dashboard polls for telemetry data.
